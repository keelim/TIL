# 최소 공배수와 최대 공약수

가끔 최대 공약수와 최소 공배수를 이용한 문제가 나오곤 합니다.
c++17 에서는 #include <numeric> 에서 stl 로 gcd 와 lcm 을 지원하니 참고하시면 됩니다.

```cpp
int gcd(int a, int b){
  return b ? Euclidean(b, a%b) : a;
}
```

```cpp
int gcd(int a, int b){
 return b? gcd(b, a%b) : a;
}

int lcm(int a, int b){
 return a*b/gcd(a, b);
}
```
